Definição de Logaritmo

O logaritmo é a operação matemática que determina o expoente que uma base deve ter para resultar num determinado número.

Se by = x, então logb(x) = y

Onde:

  • b é a base do logaritmo (b > 0 e b ≠ 1)
  • x é o resultado do expoente (x > 0)
  • y é o logaritmo

Tipos Principais de Logaritmos

Logaritmo Natural (ln)

Base e ≈ 2,71828...

ln(x) = loge(x)

Logaritmo Decimal

Base 10

log(x) = log10(x)

Logaritmo Binário

Base 2

log2(x)

Propriedades dos Logaritmos

Propriedades Fundamentais

logb(1) = 0
logb(b) = 1
blogb(x) = x

Domínio e Imagem

Domínio: x > 0

Imagem: ℝ (todos os números reais)

Gráfico da Função Logarítmica

2.0

Regras Operatórias

Produto

logb(x · y) = logb(x) + logb(y)

Exemplo:

log(20) = log(4 · 5) = log(4) + log(5)

Quociente

logb(x/y) = logb(x) - logb(y)

Exemplo:

log(8/2) = log(8) - log(2)

Potência

logb(xn) = n · logb(x)

Exemplo:

log(2³) = 3 · log(2)

Mudança de Base

logb(x) = logc(x) / logc(b)

Exemplo:

log2(8) = ln(8) / ln(2)

Calculadora de Logaritmos

Exemplos Resolvidos

Exemplo 1: Propriedade do Produto

Calcular log(12) usando a propriedade do produto

  1. log(12) = log(3 · 4)
  2. = log(3) + log(4)
  3. ≈ 0,477 + 0,602
  4. ≈ 1,079

Exemplo 2: Mudança de Base

Calcular log₂(8) usando mudança de base

  1. log₂(8) = ln(8) / ln(2)
  2. = 2,0794 / 0,6931
  3. = 3

Exercícios Práticos

Exercício 1

Calcule o valor de x na equação:

log₂(x) + log₂(x+1) = 3
  1. Usando a propriedade do produto:
    log₂(x(x+1)) = 3
  2. Aplicando a definição:
    x(x+1) = 2³ = 8
  3. Resolvendo a equação quadrática:
    x² + x - 8 = 0
  4. x = 2 (x = -4 não pertence ao domínio)

Exercício 2

Simplifique a expressão:

log₃(27) + log₃(9) - log₃(3)
  1. log₃(27) = 3 (pois 3³ = 27)
  2. log₃(9) = 2 (pois 3² = 9)
  3. log₃(3) = 1
  4. 3 + 2 - 1 = 4
